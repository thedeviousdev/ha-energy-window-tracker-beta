# Energy Window Tracker (Beta)

Home Assistant custom integration for tracking energy usage by named time windows, with a windows-based setup flow.

## Install (manual)

1. Copy `custom_components/energy_window_tracker_beta` into your Home Assistant config directory under `custom_components/`.
2. Restart Home Assistant.
3. Go to **Settings -> Devices & Services -> Add Integration**.
4. Search for **Energy Window Tracker (Beta)**.

## HACS

This repository is structured for HACS custom integration installation.
